# MNIST MLP Classifier

Workshop assignment for the **2026 Machine Learning course**.

## Project Overview

This project builds, trains, fine-tunes, and evaluates a **Multilayer Perceptron (MLP)** classifier on the **MNIST** dataset of handwritten digits. The goal is to classify grayscale digit images from **0 to 9** using a neural network approach.

The project was implemented in **Jupyter Notebook** using **Miniconda**, **TensorFlow**, and **Keras**. The workflow includes data inspection, preprocessing, baseline model training, model fine-tuning with techniques such as **dropout** and **early stopping**, and final evaluation on the test set.
